# House Price Prediction using Machine Learning

## Overview

This project focuses on building a Machine Learning model to predict house prices based on various features such as income, housing characteristics, and location-based attributes. The project follows a complete ML pipeline including data preprocessing, visualization, feature engineering, and model building using Scikit-learn.

---

## Features

* End-to-end Machine Learning pipeline
* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature scaling and transformation
* Handling categorical variables
* Model training and evaluation
* Pipeline automation using Scikit-learn
